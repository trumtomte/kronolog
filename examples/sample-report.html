Introduction

This report documents the findings of a digital forensic investigation regarding an unauthorized intrusion detected in the KeepNote application environment. The investigation revealed a threat actor exploited multiple vulnerabilities (CVE-2023-49103, CVE-2023-49104, CVE-2023-49105) in the KeepNote application to conduct reconnaissance, gain unauthorized access, and perform unauthorized file operations. The following timeline reconstruction presents the key events of the incident based on available system logs.
